# Aptos RAG with Shelby Hot Storage

Decentralized RAG AI application on Aptos using Shelby Protocol for sub-second hot storage of datasets and model weights.

- Sub-second reads for real-time inference
- Pay-per-read micropayments
- Verifiable provenance via cryptographic proofs

Early stage: Applying for Shelby Early Access. Planning SDK integration.

Tech: TypeScript, @aptos-labs/ts-sdk, @shelby-protocol/sdk
